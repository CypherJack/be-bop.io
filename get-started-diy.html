Install be-BOP in a few minutes

To setup be-BOP, you need to  :

  • A domain : the domain name used to access your be-BOP.
  • An e-mail addresse : Only used to register on Let's Encrypt (SSL/TLS certificats).
These informations are not sent to our website : they are only used to generate the following command that you'll execute on your machine.

 

bash <(curl -fsSL https://get.be-bop.io/install.sh)

Once the command has been executed, the installer will automatically configure your environment and deploy be-BOP. You will then be able to access the web interface via your domain.
